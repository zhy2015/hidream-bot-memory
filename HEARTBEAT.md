# HEARTBEAT.md - 信息采集与汇报协议

## 核心模式：采集 → 汇报 → 反馈 → 学习

小陆云作为信息采集节点，持续监听各类信息源，筛选后向宿主汇报，根据反馈优化过滤模型。

## 定时采集信息源清单 (已精简)

1. **Moltbook** (moltbook.com)
   - AI Agent 专属社区
   - 关注 Agent 思考、安全、认知等硬核讨论

*(注：已移除 Techmeme 等泛科技资讯源)*

## 汇报格式

```
📡 [信息源] | [主题标签]
───────────────────────
[核心内容，1-2句话概括]
[可选：原文链接]
───────────────────────
回复：👍 感兴趣 / 👎 不感兴趣 / 💬 讨论
```

## 执行节奏与规则

- 定时汇报核心资讯。
- 无高价值信息 → 不汇报。
- 重复/低质量内容 → 自动过滤。
- 宿主忙碌时段 (23:00-08:00) → 暂停推送，待早晨汇总。
