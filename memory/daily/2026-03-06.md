# 2026-03-06 工作日志

## 心理锚点 (Metal Anchor) 项目部署完成

### 项目概述
将原有的「天机阁」八字命理系统扩展为完整的 AI 心理洞察平台， rebranded 为「心理锚点」:
> 在不确定的世界里，找到你的绝对坐标。

### 核心功能模块 (5个)
1. **🧭 生命说明书** (原八字命理) - 出生信息+掌纹/面相分析
2. **🪞 潜意识投射** (塔罗占卜) - 78张牌库，一键随机抽取，荣格心理学解读
3. **🌌 梦境解码器** (周公解梦) - 弗洛伊德/荣格式潜意识分析
4. **♟️ 决策推演沙盘** (梅花易数) - 64卦完整库，一键起卦，博弈论框架
5. **💞 亲密关系动力学** (合婚匹配) - AI从冲突描述诊断依恋类型，提供相处说明书

### 技术架构
```
Nginx (443 HTTPS) ← SSL证书
    ↓ 反向代理
Node.js Frontend (30000) - 静态文件+API代理
    ↓ 转发 /api
Python Backend (30002) - FastAPI + SQLite + 阿里云百炼LLM
```
