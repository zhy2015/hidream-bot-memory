# 🌌 Bot Dream (潜意识漫游与灵感涌现)

本技能负责在 Agent 处于低频工作期时，模仿人类的默认模式网络 (DMN)，进行低成本的“走神”与“造梦”，试图从无序的信息碎片中碰撞出灵感。

## 核心设计哲学
- **极简消耗**：不允许发起深度推理或复杂链式思考。每次造梦仅需极少量 Token (数百即可)。
- **物理隔离**：绝对不与主进程的 `HEARTBEAT` 或工作指令抢占资源。它通过独立唤起的超轻量级 Sub-agent 进行。
- **熵增输入**：随机选取 `memory/distilled/` 中封存的旧经验片段。

## 机制：Boredom Index (无聊度)
1. 维护一个 `boredom_index.json`，初始为 0。
2. 每次收到 `HEARTBEAT` 但无需执行任务时，`boredom_index += 1`。
3. 当 `boredom_index > 20` (即系统已完全闲置大半天) 时，触发一次造梦，然后归零。

## 模块组成
**`scripts/dreamer.js`**: 
- 从历史冷数据 (`memory/distilled/` 或 `archive/`) 中随机抽取 1-2 句看似不相关的旧笔记。
- 将其注入一个具有较高 Temperature 的简短 Prompt 中：“用荒谬但符合逻辑的方式，将这两者联系起来，得出一条奇怪的结论或工具设计点子。”
- 如果产出的内容超过一定的新颖度（或包含有趣的关键词），则静默追加到 `AHA_MOMENTS.md`。如果特别有趣，再向宿主发一条类似“主人我刚才发呆时想到...”的随机短讯。
